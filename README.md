## Gulp instructions (NPM way)
```
# install gulp-cli globally (windows client)
npm install gulp-cli -g     # shortcut for globally
npm install gulp -D         # shortcut for --save-dev
npm install --save-dev gulp-util event-stream gulp-sass gulp-autoprefixer gulp-jshint gulp-coffee gulp-clean gulp-connect gulp-browserify gulp-usemin gulp-imagemin gulp-rename
```

## Gulp instructions (Yarn way)
```
npm install gulp-cli -g
npm install gulp -D

npm install
```
